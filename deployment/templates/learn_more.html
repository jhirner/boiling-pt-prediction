Boiling Point Predictor | Learn More

  • Predictions are generated via a machine learning model using the XGBoost algorithm..
  • The model was trained using 6.1k molecules containing C, H, O, and N using data from the 2015 edition of the Yaws Handbook.
  • As a result, the model is most accurate for molecules similar to those found in the training data set. This set does not include many polyaromatics or many molecules with several hydrogen bond donors, so predictions for these classes are comparatively weak. If your molecule is likely to return an inaccurate prediction, the model will let you know.
  • This predictive model has been validated using 1.3k molecules. In the validation set, 90% of predictied values fall within ± 9.7 °C of the measured value, and 95% of predictions fall within ± 14.5 °C.
