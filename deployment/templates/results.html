About this prediction:

  • Boiling points are reported at atmospheric pressure and are valid only for molecules with boiling points in the range of 400 - 700 K (127 - 427 °C).
  • Predictions are generated via an XGBoost regression machine learning model.
  • The model was trained using 6.1k molecules containing C, H, O, and N using data from the 2015 edition of the Yaws Handbook.
  • The model has been validated using an additional 1.3k molecules. Predicted values are within +/- 10% of the measured value for 99.5 % of the validation data). Further, 92 % of the predictions are with in 10 K of the measured boiling point.
